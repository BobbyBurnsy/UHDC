# ⚠️ Unified Help Desk Console (UHDC) is now Service Desk Advanced (SDA)

Welcome to the legacy repository for the Unified Help Desk Console. 

As this tool has evolved from a simple PowerShell utility into a comprehensive interface for managing help desk operations, the name "UHDC" began to feel a bit too technical and narrow. To better reflect the professional, feature-rich experience the tool now provides, we have officially rebranded to **Service Desk Advanced (SDA)**.

> *"Service Desk Advanced isn't just a utility; it's a platform built to handle the complexities of modern IT support."*

## What's New (Besides the Name)?

While the core mission remains the same—streamlining your workflow and enabling first-touch Live-Call Remediation—this rebrand marks a definitive shift toward a highly polished, "enterprise-grade" feel. 

* **Enterprise-Grade Architecture:** We have fully committed to a foundationally secure, 100% agentless telemetry model. This drastically reduces your network's attack surface and completely eliminates endpoint agent maintenance.
* **Refined Interface:** A beautiful, user-friendly graphical dashboard that seamlessly bridges the gap between Microsoft Intune and Active Directory, making the administration of individual assets faster and easier than ever before.
* **Continued Integration:** Deeper, more resilient API connections with Microsoft Graph, Entra ID, and native Windows management protocols.
* **Streamlined Documentation:** Completely overhauled deployment guides, security architecture assessments, and a dedicated support portal designed for IT professionals.

## Where to find Service Desk Advanced

This repository is now archived and will no longer receive updates. All future development, releases, and documentation have been moved to our new official domains.

* **Official Website:** [www.servicedeskadvanced.com](https://www.servicedeskadvanced.com)
